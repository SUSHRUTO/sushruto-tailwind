Sushruto Tailwind

A lightweight utility-first CSS engine. Build UI using chai-* classes without writing CSS.

📦 Installation

npm install sushruto-tailwind-css

Or via CDN:

https://cdn.jsdelivr.net/npm/sushruto-tailwind-css/src/index.js

⚡ Usage

<script type="module"> import { sushrutoTailwind } from "sushruto-tailwind-css"; sushrutoTailwind(); </script>

🎨 Colors

  1. chai-bg-red → background: red
  2. chai-bg-blue → background: blue
  3. chai-bg-green → background: green
  4. chai-text-white → text: white
  5. chai-text-black → text: black

📏 Spacing

  1. chai-p-10 → padding: 10px
  2. chai-p-20 → padding: 20px
  3. chai-p-50 → padding: 50px
  4. chai-m-10 → margin: 10px
  5. chai-m-20 → margin: 20px

🔤 Typography

  1. chai-text-16 → font-size: 16px
  2. chai-text-24 → font-size: 24px
  3. chai-font-bold → bold text
  4. chai-text-center → center align
  5. chai-text-left → left align

📦 Layout

  1. chai-flex → display: flex
  2. chai-flex-center → center alignment
  3. chai-flex-col → column layout
  4. chai-gap-10 → spacing
  5. chai-justify-between → space between

📐 Borders

  1. chai-border-2 → border width
  2. chai-border-4 → thicker border
  3. chai-rounded-10 → border radius
  4. chai-rounded-20 → larger radius
  5. chai-border-red → colored border

🎭 Transforms

  1. chai-scale-2 → scale element
  2. chai-scale-3 → larger scale
  3. chai-rotate-45 → rotate
  4. chai-rotate-90 → rotate more
  5. chai-scale-1 → default

🖱 Hover

  1. chai-hover-scale → zoom effect
  2. chai-hover-bg-red → background change
  3. chai-hover-text-white → text change
  4. chai-hover-shadow → shadow
  5. chai-hover-border → border effect
